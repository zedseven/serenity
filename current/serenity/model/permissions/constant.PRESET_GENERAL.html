Returns a set of permissions with the original @everyone permissions set to true.

This includes the following permissions:

  • Add Reactions
  • Attach Files
  • Change Nickname
  • Connect
  • Create Invite
  • Embed Links
  • Mention Everyone
  • Read Message History
  • Read Messages
  • Send Messages
  • Send TTS Messages
  • Speak
  • Use External Emojis
  • Use VAD

Note: The Send TTS Messages permission is set to true. Consider setting this to false, via:

use serenity::model::permissions::{self, Permissions};

permissions::PRESET_GENERAL.toggle(Permissions::SEND_TTS_MESSAGES);
